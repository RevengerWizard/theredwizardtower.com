Dynamic programming is a technique for solving complex problems by breaking them down into simpler subproblems and storing the solutions to these subproblems to avoid redundant work. This approach can greatly improve the efficiency of the solution, particularly for problems with overlapping subproblems.

## Getting Started

To implement dynamic programming solutions, you need to:

1. Identify overlapping subproblems
2. Find the recurrence relation
3. Decide on memoization or tabulation
4. Implement the solution

## Memoization vs Tabulation

**Memoization** (top-down) stores results as you compute them recursively, while **Tabulation** (bottom-up) builds a table iteratively.
